Waikiki Discord Bot

This is a Discord bot used to manage my Coding Den server, but feel free to invite it to your own server as well!

Commands

command_name

  1. create-codespace - Creates a codespace for that specific user
  2. ka-publish - Publish a Khan Academy program/webpage from Discord*

Codespaces

Upon joining the private Coding Den server, this bot creates a customizable "space" for you, a channel category will you can share your projects and more!

To create a codespace, simply type /create-codespace in the #bot channel of the server. You will be given a basic "space", where you can add channels, register IoT devices, and more!

IoT Devices

This bot can also register IoT devices, which can be controlled in your codespace's dashboard channel.
